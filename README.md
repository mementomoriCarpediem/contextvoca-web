# ContextVoca Web

ContextVoca 앱 지원 웹사이트 - 랜딩페이지, 개인정보처리방침, 이용약관, 고객지원

## Pages

- `/` - 랜딩페이지 (앱 소개, 기능, 요금제)
- `/privacy` - 개인정보처리방침
- `/terms` - 이용약관
- `/support` - 고객지원 (FAQ)

## Tech Stack

- Next.js 15 (Static Export)
- TailwindCSS
- GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Deployment

`main` 브랜치에 push하면 GitHub Actions로 자동 배포됩니다.

배포 URL: https://mementomoriCarpediem.github.io/contextvoca-web/
